# 🚀 DevClub Store - Apple Products Carousel

Uma landing page futurista e interativa desenvolvida durante o treinamento da **Alura**. O projeto apresenta um carousel de produtos premium com transições suaves, animações avançadas em CSS e lógica de manipulação de DOM em JavaScript.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica e acessível para os componentes da loja.
* **CSS3**: Estilização avançada utilizando `flexbox`, animações de `keyframes`, gradientes e filtros neon para criar uma estética moderna.
* **JavaScript (ES6+)**: Lógica para controle do carousel, gestão de estados ativos, indicadores numéricos e sistema de temporizador automático.
* **Google Fonts**: Integração das fontes *Orbitron* e *Montserrat* para reforçar o estilo tecnológico.

## 🌟 Funcionalidades Principais
- **Interface Responsiva**: Design adaptado para diferentes resoluções.
- **Carousel Interativo**: Navegação manual através de botões de seta e indicadores visuais (dots).
- **Reprodução Automática**: Sistema de transição configurado para mudar o slide automaticamente a cada 5 segundos.
- **Animações Dinâmicas**: Efeitos de entrada sincronizados para textos e imagens que reagem conforme o item se torna ativo.

## 🎓 Créditos e Aprendizado
Este projeto foi desenvolvido como parte de um estudo guiado em parceria com a **Alura**. Durante o desenvolvimento, foram explorados conceitos fundamentais de Front-end, incluindo a importância do `z-index` em layouts complexos e a manipulação eficiente do DOM para criar interfaces interativas.
